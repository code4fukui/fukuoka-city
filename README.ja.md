# fukuoka-city

感染症流行警告アプリ「ワーンニング」のオープンデータを使用し、日本全国のクラウドソーシングによる感染症の症状報告をインタラクティブな地図上で可視化するウェブアプリケーションです。

## デモ

[Warning Map](https://code4fukui.github.io/fukuoka-city/warnning.html)

## 機能

- **全国マップ:** クラウドソーシングアプリ「ワーンニング」から収集された最新の日別症状報告データを、日本全国のインタラクティブな地図上に表示します。
- **報告数のインジケーター:** 場所ごとの報告数を色分けされたマーカーで示します:
    - 🔴 赤: 10件より多い報告
    - 🟡 黄色: 1〜10件の報告
    - 🔵 青: 0件の報告
- **詳細ポップアップ:** マーカーをクリックすると、その場所で報告された症状（のど、せき、発熱、鼻水）の内訳と報告日を確認できます。
- **自動ズーム:** 地図は、最新日のデータが存在するすべての場所が収まるように自動的に表示範囲を調整します。

## 使い方

このプロジェクトをローカルで実行するには、単に `warnning.html` ファイルをウェブブラウザで開いてください。ウェブサーバーやビルド手順は必要ありません。

## データ

このプロジェクトは「感染症流行警告アプリ「ワーンニング」みんなの投票データ」データセットのスナップショットを使用しており、このリポジトリに `warnning.csv` として含まれています。

オリジナルのオープンデータセットは以下から取得可能です: [https://ckan.open-governmentdata.org/dataset/2c830d40-a4f1-4335-9d05-eeaa18b765f3](https://ckan.open-governmentdata.org/dataset/2c830d40-a4f1-4335-9d05-eeaa18b765f3)
